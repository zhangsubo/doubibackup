GoFlyway 基础教程：Android 客户端使用方法

Goflyway 在新兴代理软件中还是挺不错的，特别是先后支持了 CDN、KCP方式复活被墙IP，但是只能在PC上使用就很蛋疼了，虽然作者做了个安卓客户端，但是因为是基于 Shadowsocks 安卓客户端修改而来的，所以很多人有点蒙蔽，不知道怎么用，所以我就简单给大家介绍一下 Goflyway Android 客户端使用方法。

注意：因为 Goflyway 安卓客户端目前只支持普通方式和CDN方式链接，WebSocket、KCP 方式目前还不能用。

其他 Goflyway 教程请看：https://doub.io/tag/goflyway/

另一个第三方的 Goflyway 安卓客户端大家可以试试：https://github.com/koolwiki/goflyway_android

下载安装

首先我们去 Github 下载安卓客户端（下载 aligned-signed.apk ）。

下载安装后继续。

注意：因为 Goflyway 安卓客户端是基于 Shadowsocks 安卓客户端修改而来的，所以与 SS 客户端无法共存。

使用方法

添加新账号/修改已有账号

点击展开 查看更多

打开 Goflyway 安卓客户端后，可以看到一个默认的账号，因为 Goflyway 安卓客户端是基于 Shadowsocks 安卓客户端修改而来的，所以此处的默认账户实际上是 Shadowsocks 安卓客户端自带的 SS 账号，你只要删除或者修改该默认账户，就没有广告了。

如下图所示，点击 [笔] 符号就能修改该账号了。

如下图所示，点击 [中间有个+号的纸] 符号，然后再点击 [手动设置] 就能新建一个账号了。

注意：因为是基于 Shadowsocks 安卓客户端修改而来的，所以 扫描二维码、从剪贴板导入 实际上都是只针对 SS 分享链接的，并不能使用我脚本的 Goflyway 分享链接。

普通方式连接的账号配置

如果你的账号是普通方式连接，也就是没什么 CDN、KCP、WebSocket 方式的，那么你可以继续看这个。

点击展开 查看更多

注意：作者没有删掉其他无用的选项，所以只有 服务器、远程端口、密码 是有效的，其他参数选项都不用管。

根据你的 Goflyway 账号，依次填写这三项即可。

CDN方式连接的账号配置

如果你的账号是 CDN 方式连接，那么你可以继续看这个。

点击展开 查看更多

注意：作者没有删掉其他无用的选项，所以只有 服务器、远程端口、密码 是有效的，其他参数选项都不用管。

CDN 模式就要特殊一点了，因为是基于 Shadowsocks 安卓客户端修改而来的，所以要使用 CDN 模式，需要将你的账号密码修改为：

?c域名:端口

# 例如：
# 假设你的代理服务器地址为：demo3.goflyway.pw，代理端口为：80
# 那么服务端和客户端密码修改为：
?cdemo3.goflyway.pw:80

# 注意：服务端密码也要修改成这样(包括 ?c 字符)！
# 密码包含 ?c 字符！密码包含 ?c 字符！密码包含 ?c 字符！
# 服务端修改密码为此格式后，其他客户端要使用该代理账号，肯定也要写新的密码啊。

然后依次填写 服务器、远程端口、密码 这三项即可。

分应用VPN配置

目前 Goflyway 安卓客户端只支持全局，所以路由选项只能设置为：全局。

但是 分应用 VPN 还是能用的，可以指定哪些应用走代理。

点击展开 查看更多

如下图所示，点击右侧开关开启 分应用 VPN 功能。

然后点击 分应用 VPN 这个选项(不是开关)，进入分应用 VPN 配置页面。

分应用 VPN配置页面，可以点击要走代理的应用右侧的开关，绿色代表该应用走代理。

另外，绕行模式的意思是反过来，也就是绿色代表该应用不走代理。

连接/测试账号

添加/修改完账号后，我们点击配置文件设置右侧的 [√] 对号按钮，即保存配置。

点击展开 查看更多

然后就回到了初始页面，点击选择要使用的账号（账号前面是绿色的），然后点击右下方的 圆形纸飞机 按钮，即代表启动代理。

注意：如果是第一次使用，系统往往会提示该应用要创建VPN，是否允许，允许即可。

最后，我们点击一下下方的链接信息，即可测试账号可用性。

Goflyway 安卓客户端功能还很简陋，甚至很多无用的选项都没有删除，只能希望作者多多更新了。如果你感兴趣的话，也可以去 Github 帮助作者完善 Goflyway 安卓客户端，毕竟众人拾柴火焰高！
