Goflyway 或 Brook 账号用不了？

请先去 [免费账号实时状态监控] 里看一下对应节点负载是不是 0 (负载指的是链接服务器的IP数)，如果不是那么请更新你的 Goflyway 或 Brook 客户端，包括 Goflyway Tools 或 Brook Tools 所调用的 Windows命令行版客户端！

SS/SSR 账号如何快速添加到 SS/SSR 客户端？

请使用 SS/SSR客户端，点击账号右侧的 SS 按钮 或 SSR 按钮，客户端 扫描显示的二维码 以添加账号（密码）。

如无法扫描二维码，可以复制 二维码生成页 顶部的 ss://... 或 ssr://... 链接，然后在SS/SSR客户端中通过 导入剪贴板 方式添加账号。
