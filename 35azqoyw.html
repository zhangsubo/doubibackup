代理软件

Shadowsocks — 科学上网的瑞士军刀

官方：点我进入*、GitHub

Shadowsocks是目前最稳定最好用的科学上网工具之一，你需要SS账号来使用它：免费Shadowsocks账号。

使用教程：Shadowsocks的小白使用教程

S​hadowsocks（中文名称：影梭）是一个安全的socks5代理，用于保护网络流量，是一个开源项目。​跨平台​（可运行于包括PC，Mac，手机（Android和iOS）和路由器（OpenWrt）在内的多种平台上）

ShadowsocksR(停更)

官方：Github

点击展开 查看更多

ShadowsocksR是Shadowsocks的一个客户端+服务端分支（基本已经独立），你需要SS/SSR账号来使用它：免费ShadowsocksR账号。

使用教程：ShadowsocksR的小白使用教程

S​hadowsocksR相比与Shadowsocks原版，增加了 协议/混淆 设定，在Shadowsocks原版日渐被匹配流量特征从而被运营商限速的现在，使用ShadowsocksR代理是一个不错的选择！

SSCap(停更)

官方：Github

点击展开 查看更多

SSCap是Shadowsocks的一个客户端分支，你需要SS/SSR账号来使用它：免费ShadowsocksR账号。

使用教程：支持批量检测 SS/SSR账号延迟和TCP/UDP可用性的客户端 —— SSCap

SSCap相比与Shadowsocks原版和ShadowsocksR，支持 批量检测 Shadowsocks/ShadowsocksR账号 延迟和TCP/UDP可用性等功能，并且因为系统代理设置方式异于其他Shadowsocks分支客户端，所以在网吧等不允许修改注册表的环境下，也可以正常使用！

V2ray

官方：点我进入、Github

V2Ray 是一个模块化的代理软件包，它的目标是提供常用的代理软件模块，简化网络代理软件的开发。

简单来说，它就是一个模块化代理，可以用来科学上网，支持多平台。

GoGo(停更)

官方：点我进入*

点击展开 查看更多

GoGo自带了一些代理服务器，所以安装启动客户端并设置系统代理后，就可以直接使用了。

使用教程：一个免费且快速的HTTP/SOCKS5代理软件 —— GoGotunnel

GoGo是一款基于Java开发的完全异步非阻塞的高性能代理工具, 同时支持SOCKS5和HTTP两种代理协议. GoGo使用SSL加密传输数据, 针对HTTPS网站采用隧道代理(盲转发)而非中间人方式，保证您的数据不会泄露。GoGo内置的DNS解析和地理位置解析功能, 可以让您直接访问国内外常用网站。

Brook

官方：Github

客户端使用教程：Brook代理 Windows/Android版客户端简单使用教程

客户端使用教程：『原创』更方便的Windows系统 Brook 客户端 —— Brook Tools

服务端使用教程：一个优秀的跨平台 Socks5代理软件 —— Brook 服务端教程

服务端一键脚本：『原创』一个优秀的跨平台 Socks5代理软件 —— Brook 一键安装管理脚本

Brook是一个由 Go语言编写的跨平台代理软件，支持 Linux/MacOS/Windows/Android/iOS 各个平台。

同时因为 Go语言的特性，在操作使用上非常简单。

Goflyway

官方：Github

客户端+服务端使用教程：一个由Go语言编写的轻量化 HTTP Tunnel 代理工具 —— GoFlyway

可视化 客户端 使用教程：『原创』更方便的Windows系统 Goflyway 客户端 —— Goflyway Tools

服务端一键脚本：『原创』Go语言轻量化 HTTP Tunnel 代理工具 —— GoFlyway 一键安装管理脚本

其他使用姿势：Goflyway 进阶教程：免费域名+免费CDN+HTTP伪装=被墙的IP继续做代理

GoFlyway 是一个由 Go语言编写的HTTP代理软件，目前支持 Linux/Windows 平台，因为HTTP的轻量化，速度优势很大。

PipeSocks(停更)

官方：点击进入、Github

点击展开 查看更多

这是一个 最新开发的 Socks5代理工具，目前属于开发初期，大家有兴趣的可以试一试。

使用教程：「让科学上网成为一种艺术」一个新的Socks5代理软件 —— PipeSocks

PipeSocks把整个科学上网的过程比喻成 Pump(泵)、Pipe(水管)、Tap(水龙头)。

泵安装在海外服务器上面(即 服务端)，水龙头安装在本地设备上(即 客户端)，而水管则是在必要的时候起到加速/匿名等作用。

dowsDNS(停更)

官方：Github

这是一个通过建立一个简单的本地DNS服务来科学上网的工具。

使用教程：一个 免费科学上网 的DNS小工具 —— dowsDNS

一键脚本：『原创』快速自建DNS服务器以 科学上网+屏蔽广告 —— dowsDNS 一键脚本

dowsDNS利用SNI代理等，在本地创建一个简单的DNS服务，可以通过修改设备的DNS为 127.0.0.1 来实现科学上网。

萤火虫代理（Firefly proxy）

官方：点击进入*、Github

这是一个很早就有的代理软件，到现在也一直坚持更新，虽然速度一般，但毕竟是免费的，可以做备用。

一个帮助你翻越防火墙的代理软件，开源项目。

赛风3（Psiphon3）

官方：点击进入*

赛风或许不快，但是一直都能用！安卓版的看这里：安卓科学上网合集

赛风是一个规避审查的工具—它的设计目的是访问开放的互联网，绕过审核和防火墙。他是开源并且开发于加拿大多伦多市。

XX-Net

官方：Github

XX-Net你可以将它当作是一个已经配置好并可视化易操作的GoAgent。

他的缺点是和任何基于GAE或者说GoAgent的软件一样都是依赖于谷歌IP，而谷歌IP现在几乎快封完了，没剩下多少好IP了，所以速度看脸。

  • 集成搜索Google IP；
  • 基于Web 的配置界面、上传部署、Log查看，也就是说你的所有配置都可以在网页上进行，比较直观；
  • 内置了公共 appid, 安装即可上网, 当然推荐注册自己的appid；
  • 新版本提示、更新升级。

GoAgent现在的位置很尴尬，IP被封杀的很快，你扫IP的速度还有可能追不上封杀的速度，而且很那扫到优质IP了。想用的还是试试XX-Net吧！

洋葱浏览器（Tor Browser）

官方：点击进入*

由Mozilla Firefox ESR浏览器制作的，配合Tor代理，可匿名访问网络！

注意：洋葱浏览器只是一个匿名上网的浏览器，主要是保护隐身、防止追踪等安全保护的，并不带科学上网功能，需要配合其他代理软件使用。

洋葱浏览器（Tor Browser）前身为 Tor Browser Bundle (TBB), 是Tor项目的旗舰产品。由Mozilla Firefox ESR 浏览器修改而成，预载TorButton, TorLauncher, NoScript 和 HTTPS Everywhere 等扩充套件与Tor代理，无需安装的绿色软件。

Tor Browser会自动通过Tor网络启动Tor的后台进程连​接网络。一旦关闭程序的便会自动删除隐私敏感数据，如HTTP cookie和浏览历史记录。

Tor浏览器套件是一个免费的计算机应用程式，可以下载和使用，以避 开 互联网过滤。它可用来遮蔽一个互联网用户的IP地址，允许匿名浏览。

Tor浏览器套件是提供包括简体字（无繁体中文）在内的多国语言界面。
